Simple Encryption and Decryption

Encrypting with the Caesar Cipher
This project uses Python to encrypt and decrypt messages through the Caesar Cipher method. The idea of the Caesar Cipher is to pick an integer and shift every letter of your message by that integer. The user provides a Plain text Message with an integer, which can be encrypted by using the PlaintextMessage function; the user can also provide an Encrypted text Message, and the CiphertextMessage function can be used to help decrypt.

• Skills used: for loops, while loops, functions, String/list operations, Assertions/Exceptions, Classes/Inheritance, Dictionaries, module import, getter/setter

Scrabble game

Scrabble with your computer (or SkyNet if you like)
This project uses Python to write a small game called Scrabble. The computer generates a list of letters, and the user can enter words made up of these letters. The computer will check whether the word is valid and give a corresponding score. If the user use all letters on the first try, the user will get 50 bonus points! ! The game allows the user to play an arbitrary number of hands. The user can choose to play a new (random) hand or play the last hand again.

An updated version to enable your computer (aka SkyNet) to play the game. The user will be able to compare how you as a user succeed in the game compared to the computer's performance.
